# ML Learning Journey

A production-ready, data-driven, gamified roadmap for every project listed on Deep-ML Projects as captured on July 12, 2026.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Add another project

Add one object to `lib/projects.ts`. The roadmap node, unlock sequence, explorer card, search/filter results, dashboard totals, rewards, static detail route, and blog/resource links update automatically.

## Progress model

Completion, notes, bookmarks, activity, XP, level, streak, coins, stars, and the learning heatmap are stored in browser local storage under `ml-learning-journey-v1`. GitHub integration is presented as a connection-ready state; it intentionally does not use fabricated repository data.
